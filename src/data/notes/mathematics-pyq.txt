Q. Domain of the real valued function f(x) = log(x^2 - 10x + 9) is:
A) (9, infinity)
B) (-infinity, 1)
C) (1, 9)
D) (-infinity, 1) U (9, infinity)
Answer: D
Explanation: x^2 - 10x + 9 > 0 means (x-1)(x-9) > 0, so x < 1 or x > 9. Domain is (-infinity,1) U (9,infinity).

Q. For what value of a real number p, the quadratic equation px^2 - 2x + 3 = 0 lies above the x axis?
A) p < 1/3
B) p > 1/3
C) p = 1/3
D) p <= 1/3
Answer: B
Explanation: For parabola to lie above x-axis, discriminant < 0 and p > 0. D = 4 - 12p < 0 gives p > 1/3.

Q. The number of terms in the expansion of (a + b + c)^10 is:
A) 45
B) 55
C) 66
D) 78
Answer: C
Explanation: Number of terms in (a+b+c)^n = C(n+2, 2) = C(12,2) = 66.

Q. The sum of first n terms of an AP is 4n^2 + 3n. Then the 10th term is:
A) 80
B) 78
C) 77
D) 79
Answer: D
Explanation: T_n = S_n - S_(n-1) = 8n - 1. T_10 = 8(10) - 1 = 79.

Q. The sum of all three-digit numbers (repetition of digits not allowed) formed using the digits 1, 2, 3, 4 is:
A) 6660
B) 1110
C) 5550
D) 4440
Answer: A
Explanation: Each digit appears in each position (3!/1) / 4 * 3 = 6 times across all numbers. Sum = 6(1+2+3+4)(100+10+1) = 6*10*111 = 6660.

Q. Value of the expression 1/sin10 - sqrt(3)/cos10 is:
A) 1
B) 2
C) 3
D) 4
Answer: D
Explanation: Simplifying using compound angle formula: (cos10 - sqrt(3)sin10)/(sin10*cos10) = 2cos70/(sin20/2) = 4. Answer is 4.

Q. The area enclosed by the line segment joining the points (1,2), (3,4) with the coordinate axes is (sq. units):
A) 2
B) 1
C) 1/2
D) 1/4
Answer: B
Explanation: The area of the triangle formed by the line joining (1,2) and (3,4) with coordinate axes is 1 sq unit.

Q. Let G be a square matrix of order 4 with |G| = 10. Then |adj(G)| is:
A) 10^4
B) 10^3
C) 10^2
D) 10
Answer: B
Explanation: For a matrix of order n, |adj(A)| = |A|^(n-1). Here n=4, |G|=10, so |adj(G)| = 10^3 = 1000.

Q. Which among the following data set is consistent?
A) (10, 20)
B) (5, 15)
C) (11, 9)
D) (8, 12)
Answer: A
Explanation: Consistency refers to lower coefficient of variation. (10,20) has the lowest relative spread making it the most consistent dataset.

Q. Let a, b, c, d be non-negative real numbers with a+b+c+d=1. Then the maximum value of (1-a)(1-b)(1-c)(1-d) is:
A) (3/4)^4
B) (1/2)^4
C) (2/3)^4
D) 1
Answer: A
Explanation: By AM-GM inequality, when a=b=c=d=1/4, the expression is maximized = (3/4)^4.

Q. Let the perimeter of a rectangle be given as 40 cm. Then the maximum area (sq.cm) of the rectangle is:
A) 96
B) 99
C) 100
D) 75
Answer: C
Explanation: Perimeter = 2(l+b) = 40, so l+b = 20. Area is maximum when l=b=10. Max area = 10x10 = 100 sq.cm.

Q. The value of the integral from 0 to 2 of |x^2 - x| dx is:
A) 8/3
B) 7/3
C) 5/3
D) 1
Answer: D
Explanation: x^2−x changes sign at x=1, so split the integral into two parts: from 0 to 1 and from 1 to 2. Evaluating both parts gives 1/6 and 5/6. Adding them, the value of the integral is 1.

Q. The probability that a leap year will have 53 Sundays and 53 Mondays is:
A) 2/7
B) 3/7
C) 1/7
D) 4/7
Answer: C
Explanation: A leap year has 366 days = 52 complete weeks + 2 extra days. Only the pair {Sun,Mon} gives both 53 Sundays and 53 Mondays. Probability = 1/7.

Q. Out of 21 students appeared in a test only 11 have passed and their marks are 50, 49, 47, 46, 44, 47, 52, 48, 51, 55, 54. The median mark of 21 students is:
A) 40
B) 44
C) 47
D) 46
Answer: B
Explanation: Out of 21 students, 10 failed and therefore scored less than 44 (the lowest passing mark). When all 21 marks are arranged in ascending order, the first 10 positions are occupied by the failed students, making 44 the 11th observation. Since the median of 21 observations is the 11th observation, the median mark is 44.

Q. There are two variables X and Y. X takes values 1,2,...,10 and Y takes values 101,102,...,110. Which of the following is true in the case of standard deviation?
A) SD(X) < SD(Y)
B) SD(X) > SD(Y)
C) SD(X) = SD(Y)
D) None of these
Answer: C
Explanation: Standard deviation measures spread, not absolute values. Y = X + 100, so SD(Y) = SD(X). They are equal.

Q. For what value of theta, the vectors i+2j+3k, -i+5j-theta*k, i-2k are coplanar vectors?
A) 29/2
B) 29/3
C) -29/3
D) -29/2

A) a.b / |a|
B) a.b / |b|

Q. If i = sqrt(-1), then the value of i^i is:
A) e^(-pi/2)
B) e^(pi/2)
C) pi/2
D) -pi/2
Answer: A
Explanation: i^i = e^(i*ln(i)) = e^(i * i*pi/2) = e^(-pi/2).

Q. A fair coin is tossed thrice. Expected number of heads obtained is:
A) 7/2
B) 5/2
C) 9/2
D) 3/2
Answer: D
Explanation: For n coin tosses, E(heads) = n * p = 3 * 1/2 = 3/2.

Q. The solution of the differential equation dy/dx + y/x = log x, given that y(0) = 0 is:
A) y = x/4 (2 logx - 1)
B) y = x/4 (1 - 2 logx)
C) y = x/2 (logx - 1)
D) y = x/2 (logx + 1)
Answer: A
Explanation: This is a linear first-order ODE. Integrating factor = x. Solving and applying initial condition gives y = x/4(2logx - 1).

Q. A square is chosen at random from a chess board of dimension 8x8. The probability that the chosen square is of dimension 6 units is:
A) 1/64
B) 3/68
C) 3/17
D) 2/17
Answer: B
Explanation: Number of 6×6 squares on an 8×8 chessboard = (8−6+1)^2=9. Total number of squares on an 8×8 chessboard = 1^2+2^2+⋯+8^2=204. Probability = 9/204=3/68. Therefore, the answer is 3/68.

Q. If x is any positive real number, then the equation cos(theta) = x + 1/x has how many real solutions?
A) unique
B) infinite
C) finite
D) no
Answer: D
Explanation: By AM-GM, x + 1/x >= 2 for x > 0. But cos(theta) <= 1, so the equation has no real solution.

Q. The number of distinct elements in a skew symmetric matrix of order p is:
A) p(p-1)/2
B) p(p+1)/2
C) p^2 + p + 1
D) p^2 - p + 1
Answer: A
Explanation: In a skew symmetric matrix, all diagonal elements are 0 and the elements below the main diagonal are determined by the corresponding elements above it. Hence, only the elements above the main diagonal are independent. Number of distinct elements = p(p−1)/2. Therefore, the answer is p(p−1)/2.

Q. There are 4 letters and 4 addressed envelopes. In how many ways can these letters be put into four envelopes so that only three letters go into correct envelopes?
A) 3
B) 6
C) 0
D) 9
Answer: C
Explanation: If exactly 3 letters are in correct envelopes, the 4th must also be in its correct envelope (only one envelope left). So it is impossible to have exactly 3 correct. Answer is 0.
