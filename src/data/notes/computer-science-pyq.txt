LBS MCA ENTRANCE EXAM 2025 - COMPUTER SCIENCE QUESTIONS

Q1. The language used to define database structure:
A) DML
B) DDL
C) DCL
D) TCL
Answer: B
Explanation: DDL (Data Definition Language) is used to define and modify database structure using commands like CREATE, ALTER, DROP.

Q2. The SQL command used to remove a table from a database:
A) DELETE
B) REMOVE
C) DROP
D) ERASE
Answer: C
Explanation: DROP TABLE removes the entire table structure and its data from the database permanently.

Q3. Which of the following is a valid SQL constraint?
A) LIMIT
B) BLOCK
C) UNIQUE
D) FIXED
Answer: C
Explanation: UNIQUE is a valid SQL constraint that ensures all values in a column are different.

Q4. What is a primary key?
A) Duplicate field
B) Field with NULLs
C) Unique identifier
D) None of these
Answer: C
Explanation: A primary key is a unique identifier for each record in a table; it cannot be NULL or duplicate.

Q5. Which of the following is not a type of JOIN?
A) INNER
B) OUTER
C) LEFT
D) UPPER
Answer: D
Explanation: UPPER is not a JOIN type. Valid JOIN types include INNER, OUTER, LEFT, RIGHT, and FULL.

Q6. What is normalization?
A) Data duplication
B) Organizing data efficiently
C) Data sorting
D) Data encryption
Answer: B
Explanation: Normalization is the process of organizing data in a database to reduce redundancy and improve data integrity.

Q7. Which function returns the number of rows in SQL?
A) SUM()
B) COUNT()
C) AVG()
D) NUMBER()
Answer: B
Explanation: COUNT() returns the number of rows that match a specified condition in SQL.

Q8. The command used to give permission to users in SQL:
A) GRANT
B) ALLOW
C) PERMIT
D) GIVE
Answer: A
Explanation: GRANT is the DCL command used to give specific privileges to database users.

Q9. A database is a collection of:
A) Files
B) Tables
C) Related data
D) Schemas
Answer: C
Explanation: A database is an organized collection of related data stored and accessed electronically.

Q10. What is a foreign key?
A) Unique field
B) Primary key of another table
C) Random value
D) NULL
Answer: B
Explanation: A foreign key is a field in one table that refers to the primary key in another table, establishing a relationship.

Q11. The keyword used to change data in SQL is:
A) MODIFY
B) CHANGE
C) UPDATE
D) EDIT
Answer: C
Explanation: UPDATE is the SQL DML command used to modify existing records in a table.

Q12. What does ACID stand for in DBMS?
A) Atomicity, Consistency, Isolation, Durability
B) Add, Check, Integrate, Drop
C) Access, Commit, Integrate, Data
D) Access, Check, Integrate, Data
Answer: A
Explanation: ACID stands for Atomicity, Consistency, Isolation, and Durability — the four properties of reliable database transactions.

Q13. The SQL keyword which sorts the result-set:
A) GROUP BY
B) SORT
C) ORDER BY
D) ARRANGE
Answer: C
Explanation: ORDER BY is used in SQL to sort the result-set in ascending or descending order.

Q14. What is the output of the following code?
int main() {
int a = 5, b = 2;
printf("%d", a++ + ++b);
return 0;
}
A) 7
B) 8
C) 6
D) 9
Answer: B
Explanation: a++ uses a=5 first (post-increment), ++b increments b to 3 first (pre-increment). So 5+3=8. After, a becomes 6.

Q15. Which of the following is not a valid storage class in C?
A) auto
B) register
C) static
D) dynamic
Answer: D
Explanation: C has four storage classes: auto, register, static, and extern. dynamic is not a valid storage class in C.

Q16. The C library function used to release dynamically allocated memory during runtime:
A) free()
B) NULL
C) remove()
D) empty()
Answer: A
Explanation: free() is used to deallocate memory that was previously allocated using malloc(), calloc(), or realloc().

Q17. The output of the expression 5 * 2 + 3 * 4 will be:

B) 23

Q18. The operator which has the highest precedence in C:

Q19. What is a dangling pointer in C?

B) Pointer that refers to freed memory
C) Pointer pointing to another pointer
D) Pointer not declared
Answer: B
Explanation: A dangling pointer points to a memory location that has been freed or deallocated, leading to undefined behavior.

Q20. The header file needed for printf function:

A) for
B) while
C) do-while
D) None of these

Explanation: The do-while loop checks the condition after executing the body, so it always executes at least once.

B) Exits loop

D) Restarts loop

Explanation: The break statement terminates the innermost loop immediately and transfers control to the statement after the loop.
